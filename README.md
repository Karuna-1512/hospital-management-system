# 🏥 Hospital Management System

A full-stack Hospital Management System developed to manage hospital operations digitally.

## 🚀 Technologies Used

### Frontend
- React.js
- JavaScript
- Axios
- CSS

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL

## 📌 Modules

- 🔐 Admin Authentication
- 👨‍⚕️ Doctor Management
- 🧑‍🤝‍🧑 Patient Management
- 📅 Appointment Management
- 💳 Billing Management
- 📊 Dashboard
- 📄 Reports
